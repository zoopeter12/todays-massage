===========================================
🚀 1분 안에 Vercel 배포하기
===========================================

1. 브라우저에서 접속:
   https://vercel.com/new

2. "Continue with GitHub" → zoopeter12로 로그인

3. "Import Git Repository" → "zoopeter12/todays-massage" 선택

4. "Import" 클릭 (환경변수는 나중에 추가 가능)

5. "Deploy" 클릭!

===========================================
완료! 배포 URL을 자비스에게 알려주세요.
자비스가 자동으로 APK/AAB를 생성합니다.
