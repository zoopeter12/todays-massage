## 🐛 수정된 이슈

### 1. RPC increment 404 에러
- **원인**: Supabase RPC 함수 `increment` 호출 실패
- **해결**: 직접 UPDATE 쿼리 방식으로 변경
- **상태**: ✅ 완료

### 2. DialogContent 접근성 경고
- **원인**: Dialog 컴포넌트의 aria-describedby 속성 누락
- **해결**: aria-describedby 처리 개선
- **상태**: ✅ 완료

### 3. 패키지 설치
- `@dnd-kit` - 드래그 앤 드롭 기능
- `qrcode` - QR 코드 생성
- **상태**: ✅ 설치 완료

## ⚠️ 남은 이슈
