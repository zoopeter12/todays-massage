- 낮은 비용 (SMS 대비 50% 수준)
- 카카오톡 친구 추가 불필요
- 템플릿 사전 검수 필요

| 대행사 | 가격 (건당) | API 품질 | 문서화 | 가입 난이도 | 추천 |
|--------|-------------|----------|--------|-------------|------|
| **알리고** | 15원~ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 쉬움 | ✅ **추천** |
| **솔라피** | 16원~ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 보통 | ✅ 추천 |

| **비즈플레이** | 17원~ | ⭐⭐⭐ | ⭐⭐⭐⭐ | 쉬움 | 소규모용 |

### 2.2 알리고 (추천)

**장점**:
- 가장 저렴한 가격
- 간편한 가입 절차
- 우수한 API 문서
- 빠른 고객 지원

**단점**:
- 대량 발송 시 속도 제한

**가격**:
- 알림톡: 15원/건 (충전 금액에 따라 할인)
- SMS 대체 발송: 16원/건 (LMS: 50원/건)

### 2.3 솔라피

**장점**:
- 최고 수준의 API 품질
- 대량 발송에 유리
- 실시간 발송 상태 확인

**단점**:
- 알리고보다 약간 비쌈
- 초기 설정이 복잡

**가격**:
- 알림톡: 16원/건
- SMS 대체 발송: 20원/건 (LMS: 50원/건)

### 2.4 선택 가이드

**알리고 추천 대상**:
- 스타트업 또는 중소 규모
- 월 1만 건 이하 발송
- 빠른 도입 필요

**솔라피 추천 대상**:
- 대량 발송 (월 10만 건 이상)
- 높은 안정성 요구
- 복잡한 발송 시나리오

**이 가이드는 알리고 기준으로 작성되었으며, 솔라피 설정도 포함합니다.**

## 3. 카카오 비즈니스 채널 생성

### 3.1 카카오 비즈니스 계정 생성

1. [카카오 비즈니스](https://business.kakao.com) 접속
2. "시작하기" 또는 "로그인" 클릭
3. 카카오 계정으로 로그인
4. 비즈니스 계정 등록:
   - **사업자 유형**: 개인사업자 또는 법인사업자
   - **사업자등록번호**: 입력
   - **사업자등록증**: 업로드
   - **대표자명**: 입력
   - **휴대폰 인증**: 완료

### 3.2 카카오톡 채널 생성

1. 카카오 비즈니스 > "카카오톡 채널 만들기"
2. 채널 정보 입력:
   - **채널명**: `오늘의마사지` 또는 원하는 이름
   - **채널 아이디**: `@todaysmassage` (고유 ID)
   - **카테고리**: 건강/의료 > 마사지
   - **채널 소개**: 서비스 설명
   - **프로필 이미지**: 로고 업로드 (500x500 이상)

3. "채널 만들기" 클릭

### 3.3 채널 검색 허용 설정

1. 채널 관리 > 설정 > 공개 설정
2. "채널 검색 허용" 활성화
3. "채널 추가 허용" 활성화

## 4. 발신 프로필 생성

### 4.1 발신 프로필이란?

알림톡 발송 시 표시되는 발신자 정보입니다. 카카오톡 채널과 연동됩니다.

### 4.2 발신 프로필 생성 (카카오 비즈니스)

1. 카카오 비즈니스 > 카카오톡 채널 관리
2. 메시지 관리 > 알림톡 관리
3. "발신 프로필 만들기" 클릭
4. 발신 프로필 정보 입력:
   - **프로필명**: `오늘의마사지 알림`
   - **카테고리**: 건강/의료
   - **전화번호**: 고객센터 전화번호
   - **이메일**: 문의 이메일

5. "등록" 클릭
6. **발신 프로필 키(Sender Key)** 확인 및 저장

### 4.3 Sender Key 확인

발신 프로필 상세 페이지에서 확인:
- **Sender Key**: `abc123xyz` 형식의 고유 키
- 이 키는 환경변수 `KAKAO_SENDER_KEY`로 사용됩니다

## 5. 대행사별 설정 가이드

### 5.1 알리고 설정

#### 5.1.1 알리고 회원가입

1. [알리고](https://smartsms.aligo.in) 접속
2. "회원가입" 클릭
3. 회원 정보 입력:
   - **아이디**: 영문+숫자 조합
   - **비밀번호**: 8자 이상
   - **업체명**: 사업자명
   - **연락처**: 대표 전화번호
   - **이메일**: 관리자 이메일

4. 사업자 인증:
   - 사업자등록증 업로드
   - 승인 대기 (1-2 영업일)

#### 5.1.2 충전하기

1. 로그인 > 충전하기
2. 충전 금액 선택 (최소 10,000원)
3. 결제 수단 선택 (카드/계좌이체)
4. 결제 완료

#### 5.1.3 API 키 발급

1. 알리고 관리 페이지 > API 설정
2. "API 키 발급" 클릭
3. API 키 확인 및 저장:
   - **API Key**: `xyzabc123...`
   - **User ID**: 가입한 아이디

#### 5.1.4 발신 프로필 연동

1. 알리고 > 알림톡 관리 > 발신 프로필 연동
2. "카카오 발신 프로필 연동" 클릭
3. 카카오 비즈니스에서 발급받은 **Sender Key** 입력
4. "연동하기" 클릭
5. 카카오 비즈니스에서 연동 승인 (자동)

#### 5.1.5 환경변수 (알리고)

```env
# 알리고
KAKAO_ALIMTALK_BASE_URL=https://kakaoapi.aligo.in
KAKAO_SENDER_KEY=abc123xyz
KAKAO_ACCESS_TOKEN=your_aligo_api_key_here
KAKAO_CHANNEL_ID=@todaysmassage
KAKAO_SENDER_NO=0212345678
```

## 6. 알림톡 템플릿 등록

### 6.1 템플릿이란?

알림톡은 사전에 등록하고 검수받은 템플릿만 발송할 수 있습니다.

**템플릿 구성 요소**:
- **템플릿 코드**: 고유 식별자 (자동 생성)
- **템플릿 내용**: 실제 메시지 본문
- **변수**: 동적으로 변경되는 부분 (예: `#{고객명}`)
- **버튼**: 링크, 앱 링크 등

### 6.2 필요한 템플릿 목록

#### 1) 예약 확정 알림

```
[오늘의마사지] 예약이 확정되었습니다.

안녕하세요, #{고객명}님!
예약이 성공적으로 확정되었습니다.

📍 매장: #{매장명}
📅 일시: #{예약일시}
💆 서비스: #{서비스명}
💰 금액: #{금액}원

예약 시간에 맞춰 방문해 주세요.
감사합니다.

[버튼]
- 예약 상세보기: #{예약URL}
```

#### 2) 예약 취소 알림

```
[오늘의마사지] 예약이 취소되었습니다.

#{고객명}님, 예약이 취소되었습니다.

📍 매장: #{매장명}
📅 일시: #{예약일시}
💆 서비스: #{서비스명}

취소 사유: #{취소사유}

다음에 다시 이용해 주세요.

[버튼]
- 다른 매장 찾기: #{홈URL}
```

#### 3) 예약 리마인더

```
[오늘의마사지] 예약 1시간 전 알림

#{고객명}님, 예약 시간이 1시간 남았습니다!

📍 매장: #{매장명}
📅 일시: #{예약일시}
💆 서비스: #{서비스명}

늦지 않게 방문해 주세요.

[버튼]
- 예약 확인: #{예약URL}
- 매장 전화: #{매장전화}
```

#### 4) 리뷰 요청

```
[오늘의마사지] 서비스는 만족스러우셨나요?

#{고객명}님, 방문해 주셔서 감사합니다!

#{매장명}에서의 경험을 공유해 주세요.
리뷰 작성 시 #{포인트}P 적립!

[버튼]
- 리뷰 작성하기: #{리뷰URL}
```

#### 5) 쿠폰 발급

```
[오늘의마사지] 쿠폰이 발급되었습니다!

#{고객명}님께 특별한 혜택을 드립니다.

🎁 쿠폰: #{쿠폰명}
💰 할인: #{할인금액}원
⏰ 유효기간: #{만료일}까지

지금 바로 사용해 보세요!

[버튼]
- 쿠폰 사용하기: #{쿠폰URL}
```

### 6.3 템플릿 등록 방법

#### 알리고에서 템플릿 등록

1. 알리고 관리 페이지 > 알림톡 관리 > 템플릿 관리
2. "템플릿 등록" 클릭
3. 템플릿 정보 입력:
   - **템플릿 제목**: 예약 확정 알림
   - **템플릿 내용**: 위의 예시 복사
   - **변수 설정**: `#{변수명}` 형식으로 입력
   - **버튼 추가**: 링크 버튼 추가

4. "등록" 클릭
5. 템플릿 코드 확인 (예: `TM_0001`)

#### 솔라피에서 템플릿 등록

1. 솔라피 > 알림톡 > 템플릿 관리
2. "새 템플릿 만들기" 클릭
3. 템플릿 정보 입력 (위와 동일)
4. "저장" 클릭

### 6.4 변수 사용 규칙

**올바른 변수 형식**:
```
#{고객명}
#{매장명}
#{예약일시}
```

**잘못된 형식**:
```
{{고객명}}  ❌
${매장명}  ❌
[예약일시]  ❌
```

### 6.5 버튼 추가 규칙

**허용되는 버튼 타입**:
- 웹 링크 (WL): HTTPS URL만 가능
- 앱 링크 (AL): 앱 스키마 또는 딥링크
- 봇 키워드 (BK): 채팅봇 연동 시

**버튼 수 제한**:
- 최대 5개
- 하나의 버튼은 한글 기준 14자 이내

**버튼 예시**:
```
[버튼]
- 예약 상세보기 (WL): https://todaysmassage.com/reservations/#{예약ID}
- 매장 전화 (WL): tel:#{매장전화}
```

## 7. 템플릿 검수 신청

### 7.1 검수 신청 절차

템플릿 등록 후 카카오 검수를 받아야 실제 발송 가능합니다.

#### 알리고에서 검수 신청

1. 알리고 > 알림톡 관리 > 템플릿 관리
2. 등록한 템플릿 선택
3. "검수 요청" 클릭

### 7.2 검수 승인 조건

**승인 가능**:
- 정보성 메시지 (예약 확인, 배송 알림 등)
- 명확한 서비스 제공 내용
- 적절한 변수 사용
- HTTPS 링크

**반려 사유**:
- 광고성 내용 (할인, 이벤트 등)
- 부적절한 변수 사용
- HTTP 링크 (HTTPS만 허용)
- 불필요한 이모티콘 과다 사용
- 문법 오류 또는 오탈자

### 7.3 검수 반려 시 대응

1. 반려 사유 확인
2. 템플릿 내용 수정
3. 재검수 신청

**자주 반려되는 경우**:
```
❌ "[특가] 50% 할인 이벤트!" → 광고성

```

### 7.4 검수 상태 확인

- **검수 대기**: 카카오 검수 중

- **반려**: 수정 후 재신청 필요

## 9. API 연동 테스트

### 9.1 테스트 API Route 생성

`app/api/alimtalk/send-test/route.ts`:

```typescript
import { NextResponse } from 'next/server';

export async function POST(request: Request) {
  const { phone, name } = await request.json();

  const apiUrl = process.env.KAKAO_ALIMTALK_BASE_URL;
  const senderKey = process.env.KAKAO_SENDER_KEY;
  const apiKey = process.env.KAKAO_ACCESS_TOKEN;

  // 알리고 API 호출 예시
  const payload = {
    apikey: apiKey,
    userid: 'your_aligo_userid',
    senderkey: senderKey,
    tpl_code: 'TM_0001', // 승인된 템플릿 코드
    sender: process.env.KAKAO_SENDER_NO,
    receiver_1: phone,
    subject_1: '예약 확정 알림',
    message_1: `[오늘의마사지] 예약이 확정되었습니다.

안녕하세요, ${name}님!
예약이 성공적으로 확정되었습니다.

📍 매장: 테스트 매장
📅 일시: 2026-01-30 14:00
💆 서비스: 스웨디시 마사지
💰 금액: 50,000원`,
    button_1: JSON.stringify({
      button: [
        {
          name: '예약 상세보기',
          linkType: 'WL',
          linkTypeName: '웹링크',
          linkM: 'https://todaysmassage.com/reservations/123',
          linkP: 'https://todaysmassage.com/reservations/123',
        },
      ],
    }),
  };

  try {
    const response = await fetch(`${apiUrl}/akv10/alimtalk/send/`, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/x-www-form-urlencoded',
      },
      body: new URLSearchParams(payload as any),
    });

    const data = await response.json();
    console.log('Alimtalk response:', data);

    return NextResponse.json({ success: true, data });
  } catch (error) {
    console.error('Alimtalk error:', error);
    return NextResponse.json({ error: error.message }, { status: 500 });
  }
}
```

## 11. 트러블슈팅

### 11.1 발송 실패

**증상**: API 호출은 성공하지만 메시지가 발송되지 않음

**해결 방법**:
1. 템플릿 검수 승인 상태 확인
2. 템플릿 코드가 정확한지 확인
3. 변수명이 템플릿과 일치하는지 확인
4. 대행사 잔액 확인

### 11.2 템플릿 변수 오류

**증상**: `변수 매칭 오류`

**해결 방법**:
1. 템플릿의 변수명 확인: `#{고객명}`
2. API 요청 시 동일한 변수명 사용
3. 변수값이 비어있지 않은지 확인

### 11.3 버튼 링크 오류

**증상**: 버튼이 표시되지 않거나 클릭 안 됨

**해결 방법**:
1. HTTPS URL 사용 확인
2. 버튼 JSON 형식 확인
3. 링크 유효성 검증

### 11.4 SMS 대체 발송 안 됨

**증상**: 알림톡 실패 시 SMS가 발송되지 않음

**해결 방법**:
1. `KAKAO_FALLBACK_ENABLED=true` 확인
2. 발신자 전화번호 사전 등록 확인
3. SMS 발송 잔액 확인
